## 我的总结
页面中的各个元素大小固定，主要练习CSS中的字体、背景、颜色等属性的设置进一步，练习CSS布局

### 背影颜色透明度的设置
```CSS
  opacity:0.4;
  filter:Alpha(opacity==40);  IE
```
### 清除浮动：在浮动的元素后面插入一个空的块级元素，清除两边的浮动
```CSS
.section2:after{
	content:"";
	display:block;
	clear:both;
}
```
